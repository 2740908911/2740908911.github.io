测试一下 blog，顺便复习一下 md 语法。

测试软件为 Typora

# 一、标题

、# 这是一级标题

、## 这是二级标题

、### 这是三级标题

、#### 这是四级标题

、##### 这是五级标题

、###### 这是六级标题

# 二、字体

加粗 这是加粗文字

要加粗的文字左右分别用两个号包起来

倾斜 这是倾斜文字

要倾斜的文字左右分别用一个号包起来

斜体加粗 这是斜体加粗文字

要倾斜和加粗的文字左右分别用三个号包起来

删除线 这是加线文字

要加删除线的文字左右分别用两个～～号包起来

# 三、引用

在引用的文字前加 > 即可。引用也可以嵌套，如加两个 >> 三个 >>>

<u> 这是引用的内容 </u>

<u> 这是引用的内容 </u>

<u> 这是引用过的内容 </u>

这是引用的内容。。

# 四、分割线

三个或者三个以上的 - 或者 * 都可以。

# 五、图片

格式：![图片 alt](图片地址 '' 图片 title'')

图片 alt 就是显示在图片下面的文字，相当于对图片内容的解释。

图片 title 是图片的标题，当鼠标移到图片上时显示的内容。title 可加可不加。

示例

# 六、超链接

[超链接名] (超链接地址 "超链接 title")

去掉 ] 后的空格，title 可加可不加！

示例：

高萌的博客

# 七、列表

语法： 无序列表用 - + * 任何一种都可以

  • 列表内容 （-）

    • 列表内容 （+）

      • 列表内容 （*）

（三种列表）

列表嵌套

上一级和下一级之间敲三个空格即可

# 八、 表格

表头表头表头
内容内容内容
内容内容内容

直接从段落插入 更快一些。

# 九、代码

单行代码：代码之间分别用一个反引号包起来

(`) 不用加括号

printf("hello world\n");

代码块：代码之间分别用三个反引号包起来，且两边的反引号单独占一行

(```)，结尾同理，不用加括号
