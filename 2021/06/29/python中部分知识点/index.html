python学习中部分易忘知识点

发表于 2021-06-29
  • Yume Shoka
首页

本文是我在py学习中总结梳理了一些常用的易忘的知识点，为方便回顾和复习，遂加以记录。

本文我会不断的学习并总结。

1.不希望出现转义字符起作用，在printf()中加r/R

printf (r 'hello\n world')

#但要注意，最后一个字符不能是\

printf (r 'hello\n world\')

#上面的写法是错的

2.将数据输出到文件中去

fp = open ( 'D:/text.txt' , 'a+')

printf ('hello world' , file = fp)

fp.close()

3.相比于c语言，多的常用数据类型

布尔类型——bool ( Ture , False )

字符串类型——str ( 与c字符串类似 )

整数类型中：

printf('十进制' , )  #不用加开头

printf ('二进制' , 0b )  # 要加开头

printf ('八进制' , 0o )

printf ('十六进制' , 0x )

浮点数计算中可能出现不精确的情况，另行解决。

字符串在多换行显示用 ‘’‘xxx’‘’ or “”“xxx”“”

4.print注意

在一句printf()里,不能同时出现多种数据类型，但可以用str() / int() / float()

5.计算和定义注意

在py3里

**表示乘方

//表示整除

数字很大时，可以用下划线来分隔，更清晰易读，py不会打印下划线

相比于c语言，py可以这样赋值： x, y, z = 0, 0, 0

常量一般用全大写表示

6.总结一些常用的方法（后面遇到的也写到这里了）

title() 以首字母大写的方式显示每个单词

name = "ada lovelace"
print(name.title())
-----
Ada Lovelace

upper() 全大写字母显示单词

lower() 全小写字母显示单词

name = "ada lovelace"
print(name.upper())
print(name.lower())
-----
ADA LOVELACE
ada lovelace

format() 这个方法在3.6以后的版本已经换为用 f字符 连接字符串了（所以只写了f字符的用法）

first_name = "ada"
last_name = "lovelace"
full_name = f"{first_name} {last_name}"
print(f"Hello, {full_name.title()}!")
-----
Hello, Ada Lovelace

rstrip() 删除末尾空白

lstrip() 删除开头空白

strip() 删除字符串两边的空白

language_1 = "python "
language_2 = " python"
language_1 = language_1.rstrip()
language_2 = language_2.lstrip()
print(language_1)
print(language_2)
-----
python
python

append() 在列表末尾添加元素

insert() 在列表的任何位置添加元素

phones = ['apple', 'huawei', 'oppo', 'vivo']
phones.append('honor')
print(phones)
print()    #换行

phones = ['apple', 'huawei', 'oppo', 'vivo']
phones.insert(0,'honor')
print(phones)
-----
['apple', 'huawei', 'oppo', 'vivo', 'honor']

['honor', 'apple', 'huawei', 'oppo', 'vivo']

del语句 删除元素，需要知道元素的位置

pop() 当需要删除值并接着使用时，用pop，否则可以用del语句删除

remove() 根据值删除元素

phones = ['apple', 'huawei', 'oppo', 'vivo']
del phones[0]
print(phones)
print()    #换行

phones = ['apple', 'huawei', 'oppo', 'vivo']
buy = phones.pop(1)
print(phones)
print(f"I will buy {buy} phone.")
print()

phones = ['apple', 'huawei', 'oppo', 'vivo']
phones.remove('apple')
print(phones)
#remove 有pop一样的用法，可以在删除时调用。
#remove只删除第一次出现的值，如果有多个一样的都需要删除，需要用while循环。
-----
['huawei', 'oppo', 'vivo']

['apple', 'oppo', 'vivo']
I will buy huawei phone.

['huawei', 'oppo', 'vivo']`

7.列表（相关方法在6）

普通列表及访问

phones = ['apple', 'huawei', 'oppo', 'vivo']
print(phones)        #全部打印
print(phones[2])        #单个打印
print(phones[-1])    #索引-1可以访问最后一个

-----
['apple', 'huawei', 'oppo', 'vivo']
['oppo']
['vivo']
