字符串类型 ——str (与 c 字符串类似)

整数类型中：

# 4.print 注意

在一句 printf () 里，不能同时出现多种数据类型，但可以用 str () /int () / float ()

# 5. 计算和定义注意

在 py3 里

** 表示乘方

// 表示整除

数字很大时，可以用下划线来分隔，更清晰易读，py 不会打印下划线

相比于 c 语言，py 可以这样赋值： x, y, z = 0, 0, 0

常量一般用全大写表示

# 6. 总结一些常用的方法（后面遇到的也写到这里了）

title() 以首字母大写的方式显示每个单词

upper() 全大写字母显示单词

lower() 全小写字母显示单词

format() 这个方法在 3.6 以后的版本已经换为用 f 字符 连接字符串了（所以只写了 f 字符的用法）

rstrip() 删除末尾空白

lstrip() 删除开头空白

strip() 删除字符串两边的空白

append() 在列表末尾添加元素

insert() 在列表的任何位置添加元素

del 语句 删除元素，需要知道元素的位置

pop() 当需要删除值并接着使用时，用 pop，否则可以用 del 语句删除

remove() 根据值删除元素
