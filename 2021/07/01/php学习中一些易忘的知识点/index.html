php的浅显学习中一些知识点

发表于 2021-07-01
  • Yume Shoka
首页

为了读懂web一些漏洞和原理，浅显学习了一下php语言，并做了一点简陋的笔记，方便回顾。

后面还会不断的或是专业的学习这门语言。

1.变量以 $ 符号开始，不能包含空格，区分大小写。php定义变量，不用声明变量类型。

2.以?php 开始，以?> 结尾。

3.在函数内调用全局变量，要用 global 关键字,比如：

$x = 5;
$y = 4;
function text()
{
global $x, $y;
$x = $x + $y;
}

当然，也可这么引用：

$GLOBALS['x'] = $GLOBALS['x'] + $GLOBALS['y']

4.static作用域，用法语c语言差不多，让一个变量的值不会被清除或重置。

5.echo和print的区别：

  • echo - 可以输出一个或多个字符串，没有返回值，速度比print要快。
  • print - 只允许输出一个字符串，返回值总为 1

6.类型比较，==只比较值，不比较类型；===既比较值也比较类型

7.定义常量时 可选择是否区分大小写区分，不区分则在后面加true，例如：

define("name", "value", true);

8.并置运算符(.)，例如

echo $txt1 . " " . $txt2 ;

9.整除运算符 intdiv() ，该函数返回值为第一个参数除以第二个参数的值并向下取整数，例如intdiv(10,3) 值为3

10.赋值运算符中 a .= b,相当于 a = a . b ,连接两个字符串。

11.比较运算符中 x <> y 相当于 x != y ，x!==y会同时判断值和类型。

12.数组运算符中，x + y能拼接两个数组。

13.组合比较符<=>，用法：

$c = $a <=> $b;

if $a > $b, $c = 1;

if $a < $b, $c = -1;

if $a = $b, $c = 0;

14.数组，可以储存字符串，用 array() 函数创建。

关联数组，可以通过一个数组关联到另外的数组。

部分有关数组的函数：

  • sort() - 对数组进行升序排列
  • rsort() - 对数组进行降序排列
  • asort() - 根据关联数组的键值，对数组进行升序排列
  • ksort() - 根据关联数组的键名，对数组进行升序排列
  • arsort() - 根据关联数组的键值，对数组进行降序排列
  • krsort() - 根据关联数组的键名，对数组进行降序排列
  • count() - 用于返回数组元素的数量。

15.foreach循环，用于遍历数组。例如：

<?php 
$x = array("Google","Runoob","Taobao"); foreach ($x as $value) 
{
    echo $value . PHP_EOL; 
}
?>

16.常用函数

创建一个函数用 function xxx()

strpos() 用于在字符串内查找一个字符或一段指定的文本。

如果在字符串中找到匹配，该函数会返回第一个匹配的字符位置。如果 未找到匹配，则返回 FALSE。

17.php魔术常量

__LINE__
    #报行号
__FILE__
    #报文件路径和文件名
__DIR__
    #报文件所在目录
__FUNCTION__
    #返回函数名
__CLASS__
    #返回类名
__TRAIT__
    #trait名
__METHOD__
    #类的方法名，返回该方法被定义时的名字
__NAMESPACE__
    #当前命名空间的名称

18.php命名空间（namespace）

PHP 命名空间可以解决以下两类问题：

1.用户编写的代码与PHP内部的类/函数/常量或第三方类/函数/常量之间的名字冲突。
2.为很长的标识符名称(通常是为了缓解第一类问题而定义的)创建一个别名（或简短）的名称，提高源代码的可读性。

注意：

在声明命名空间之前唯一合法的代码是用于定义源文件编码方式的 declare 语句。所有非 PHP 代码包括空白符都不能出现在命名空间的声明之前。
